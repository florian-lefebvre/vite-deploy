# `@vite-deploy/cloudflare`

This adapter allows you to deploy your Vite project to [Cloudflare](https://www.cloudflare.com/). Learn how in our [Cloudflare Workers deployment guide](https://vite-deploy.web-runes.dev/deploy/cloudflare-workers/).

To get started, visit [the documentation](https://vite-deploy.web-runes.dev/adapters/cloudflare/).
