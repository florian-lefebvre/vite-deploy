# `@vite-deploy/netlify`

This adapter allows you to deploy your Vite project to [Netlify](https://www.netlify.com/). Learn how in our [Netlify deployment guide](https://vite-deploy.web-runes.dev/deploy/netlify/).

To get started, visit [the documentation](https://vite-deploy.web-runes.dev/adapters/netlify/).
