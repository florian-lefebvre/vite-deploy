# `@vite-deploy/node`

This adapter allows you to deploy your Vite project to hosts which support [Node.js](https://nodejs.org/en/) (and more), as well as prerender your project to get pure static files. Learn how in our [Node deployment guide](https://vite-deploy.web-runes.dev/deploy/deploy/node/).

To get started, visit [the documentation](https://vite-deploy.web-runes.dev/adapters/node/).
