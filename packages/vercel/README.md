# `@vite-deploy/vercel`

This adapter allows you to deploy your Vite project to [Vercel](https://vercel.com/). Learn how in our [Vercel deployment guide](https://vite-deploy.web-runes.dev/deploy/vercel/).

To get started, visit [the documentation](https://vite-deploy.web-runes.dev/adapters/vercel/).
