# Vite Deploy

Deploy your Vite project anywhere. The minimal primitive for shipping Vite apps to Cloudflare, Netlify, Vercel, Node servers and more. Statically, on-demand, or both.

To get started, visit [the documentation](https://vite-deploy.web-runes.dev/).
