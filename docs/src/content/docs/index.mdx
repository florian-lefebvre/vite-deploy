## Features

<CardGrid>
  <Card title="Every major host" icon="rocket">
    Dedicated [adapters](/adapters/) for Cloudflare, Netlify, and Vercel. The [Node adapter](/adapters/node/) covers Node, [Deno](/deploy/deno/), and [Bun](/deploy/bun/).
  </Card>
  <Card title="Static, server, or hybrid" icon="document">
    Prerender at build time, render on-demand at request time, or [mix both](/outputs/) in the same project.
  </Card>
  <Card title="Built on web standards" icon="setting">
    Write a `fetch` handler once, run it everywhere. Works with [Hono](/integrate-with/hono/), [Express](/integrate-with/express/), [TanStack Start](/integrate-with/tanstack-start/) and more.
  </Card>
  <Card title="Open source" icon="github">
    MIT-licensed and community-driven. [Star us, file issues, send PRs](https://github.com/web-runes/vite-deploy).
  </Card>
</CardGrid>
